# v0.1 Scope

This file is kept as historical scope. The current public package target is v0.2.

## Goal

Create the smallest useful installable package for a Codex-operated personal knowledge vault.

## Included

- Plugin manifest.
- Main Codex Skill.
- Vault template.
- Source packet workflow.
- Compilation workflow.
- Invocation workflow.
- Initialization and lint scripts.

## Not Included Yet

- Cloud sync.
- Vector database.
- Account connectors.
- Browser automation.
- Direct WeChat, DingTalk, Gmail, Slack, Notion, or Drive ingestion.
- Team permission management.

## Product Thesis

The product is not "RAG over files". The product is a repeatable operating system for converting daily work evidence into stable, traceable knowledge that Codex can call during future tasks.
