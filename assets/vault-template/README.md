# Knowledge Vault

This vault is the durable Markdown source of truth for personal, project, method, and agent knowledge.

## Principles

- Markdown is the source of truth.
- External tools collect and retrieve; they do not replace the vault.
- Raw evidence enters `00-inbox/` or `_maintenance/capture-ledger/` first.
- Stable conclusions are compiled into topic pages.
- Sensitive sources stay abstracted unless explicitly authorized.

## Reading Order

1. `全局索引.md`
2. `80-agent-briefs/knowledge-vault-entry.md`
3. `80-agent-briefs/current-focus.md`
4. Relevant project, decision, method, context, or skill pages

## Maintenance

Use `_maintenance/source-registry.md` for source governance and `80-agent-briefs/wiki-maintenance-protocol.md` for compilation rules.
