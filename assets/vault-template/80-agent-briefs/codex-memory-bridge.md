# Codex Memory Bridge

Codex memory can recover context, but it is not the source of truth.

## Write Back To The Vault When

- The user confirms a durable preference.
- A project decision affects future work.
- A method becomes reusable.
- A tool boundary or agent rule changes.
- A task creates a stable artifact or operating pattern.

## Do Not Write Back Automatically

- Temporary guesses.
- One-off execution details.
- Secrets, keys, cookies, or credentials.
- Raw private or company-confidential details.
- External facts that may have changed and were not verified.
