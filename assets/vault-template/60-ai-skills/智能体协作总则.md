# 智能体协作总则

## 分工

- Codex: local files, code, scripts, skills, vault maintenance.
- Browser or app agents: UI inspection and authenticated workflows when authorized.
- Markdown vault: durable knowledge source of truth.
- External tools: source discovery, drafting, retrieval, and temporary Q&A.

## Task Contract

Good agent requests should clarify:

- Scenario
- Goal
- Context
- Boundaries
- Output
- Done standard

## Knowledge Rule

When a task creates reusable conclusions, write them into the vault instead of leaving them only in chat.
