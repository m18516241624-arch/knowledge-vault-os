# Maintenance Plan

## Daily

- Review new source packets.
- Decide discard, log-only, inbox, compile, or review.
- Update topic pages when stable conclusions exist.

## Weekly

- Check stale active projects.
- Review uncompiled evidence.
- Merge duplicate or overlapping notes.
- Refresh agent briefs.

## Monthly

- Review source registry.
- Check sensitive-boundary handling.
- Archive inactive project pages only after explicit user approval.
